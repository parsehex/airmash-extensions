AirMash Extensions ​

These are my (and others') extensions/mods for the game AirMash. It's a fork of Fabio Spampinato's airmash-swam-extensions - please see that repository for the original code.

I don't really use StarMash so I've rewritten some extensions as well as made my own mods for the game using userscripts.

Installing Userscripts ​

You can install userscripts using a browser extension. I recommend using Violentmonkey or Tampermonkey.

Extensions ​

Respawn Hotkey ​

Adds hotkeys to make respawning faster. Press Shift + 1, 2, 3, 4, or 5 to respawn as the corresponding ships in the left corner.

STARMASH Install URL:

text
https://parsehex.github.io/airmash-extensions/extensions/respawn-hotkey.js
